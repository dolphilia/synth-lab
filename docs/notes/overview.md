# プロジェクト概要

## 目的

さまざまな音の生成手法を研究しつつ、ソフトシンセのプロトタイプを作成し、最終的に独自の音楽アプリを開発する。

## ゴールイメージ

Cycling '74 の **MAX** や BITWIG の **The Grid** をイメージしたビジュアルプログラミング環境。

- キャンバス上に音楽的なノードを自由に配置・接続できる
- ノードの組み合わせによって独特な音やリズムを生み出せる
- モジュラーシンセのような柔軟性と、ビジュアルな操作性を両立する

## 開発フェーズ

```
研究フェーズ        → 音の生成・合成・解析の手法を研究する
プロトタイプフェーズ → 個別機能や UI のプロトタイプを作成・検証する
仕様策定フェーズ    → 研究・プロトタイプの知見をもとに仕様書を整備する
アプリ開発フェーズ  → 仕様に基づいて本番アプリを開発する
```
