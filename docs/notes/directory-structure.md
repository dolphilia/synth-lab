# ディレクトリ構成

## 全体構成

```
synth-lab/
├── docs/
│   ├── notes/    調査メモ・検討メモ・雑記
│   ├── plans/    計画書
│   └── spec/     仕様書（最初はドラフトから始める）
├── prototypes/   プロトタイプ（1プロトタイプ = 1サブディレクトリ）
│   ├── <prototype-a>/
│   ├── <prototype-b>/
│   └── ...
└── research/     研究ノート・スクリプト・データ
    ├── data/     研究用の音声データ等（テーマごとにサブディレクトリで整理）
    ├── notebooks/ Jupyter Notebook（.ipynb）
    └── scripts/  Jupyter Notebook から呼び出すスクリプト
```

## 各ディレクトリの用途

### docs/

文書類を一括管理するディレクトリ。

| サブディレクトリ | 用途 |
|---|---|
| `notes/` | 調査・検討の過程で生まれたメモ。気軽に追加してよい |
| `plans/` | 機能・実装・スケジュールなどの計画書 |
| `spec/` | アプリの仕様書。草案・ドラフト状態から始め、徐々に精緻化する |

### prototypes/

個別のプロトタイプをサブディレクトリ単位で管理する。各プロトタイプは独立したプロジェクトとして扱う。

### research/

Jupyter Notebook を中心に、音の生成・合成・解析の実験・研究を行う領域。

- `notebooks/` — 実験・研究ノート本体（`.ipynb`）
- `scripts/` — ノートから `import` して使う共通スクリプト
- `data/` — 実験で生成・取得した音声データ等。テーマごとにサブディレクトリを切って整理する
