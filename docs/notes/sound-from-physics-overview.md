# 音生成に応用できる物理・数学技術 — 概要と索引

音の生成に応用できる可能性がある、シンセサイザー分野の外側にある技術・理論の調査ノート群。  
研究目的・サンプリング素材生成目的であり、計算コストは問わない。

## 計算コスト別・実装優先度マップ

### すぐに試せる（低コスト・numpy/scipy のみ）

| 手法 | 効果・特徴 |
|---|---|
| van der Pol 振動子 | リード楽器的な自励振動音 |
| Duffing 振動子 | 非線形共鳴・打楽器的「クラッシュ」音 |
| ロジスティック写像 | カオスノイズへの段階的移行 |
| ローレンツアトラクター | 複雑・カオス的波形 |
| ピンクノイズ（1/f） | 自然音に最も近い統計的ノイズ |
| 確率共鳴 | 閾値フィルター合成 |
| Hodgkin-Huxley モデル | 神経発火パターンの音化 |
| 地震波ソニフィケーション | 自然地震データの可聴化 |
| 重力波可聴化 | LIGO チャープ音（20-150 Hz） |

### 中程度の実装コスト

| 手法 | 効果・特徴 |
|---|---|
| ウェーブレット分析合成 | 多解像度の音変換 |
| フラクショナルフーリエ変換 | チャープ音・時間-周波数融合音色 |
| 薄板モーダル合成（解析解） | 太鼓・シンバルの精密モデル |
| ガンマトーンフィルタバンク | 蝸牛の周波数分析の模倣 |
| スパース合成（OMP） | 辞書ベースの音色設計 |
| 惑星磁気圏波動（公開データ） | ホイッスラー波・コーラス |

### 高コスト（研究環境・HPC 推奨）

| 手法 | 効果・特徴 |
|---|---|
| LBM による流体音響 | フルート・笛の発音シミュレーション |
| FEM 薄板振動（FEniCS） | ギター・バイオリン表板の放射音 |
| DNS 渦音 | 気流から生まれる音の完全シミュレーション |
| 量子フーリエ合成（Qiskit） | 量子確率振幅をスペクトルとして使用 |

## 主要ライブラリ一覧

| ライブラリ | 用途 |
|---|---|
| scipy.integrate（odeint/solve_ivp） | 常微分方程式（振動子・HH モデル等） |
| FEniCS / DOLFINx | 有限要素法（薄板・音響空間） |
| pylbm | 格子ボルツマン法 |
| PySpice | SPICE 回路シミュレーション |
| pywt（PyWavelets） | ウェーブレット変換 |
| gudhi | TDA（持続ホモロジー） |
| ObsPy | 地震波データ処理 |
| GWpy | 重力波データ処理 |
| Qiskit（IBM） | 量子コンピューティング |
| colorednoise | 色つきノイズ（1/f 等）生成 |
| fbm | フラクショナルブラウン運動 |
