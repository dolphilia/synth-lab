# 技術スタック

## プロトタイプ

### 基本構成

| 役割 | 技術 |
|---|---|
| ランタイム | Node.js |
| 言語 | TypeScript |
| ビルドツール | Vite |

### 追加検討

必要に応じて以下を追加する。

| 用途 | 候補 |
|---|---|
| UI フレームワーク | React |
| フルスタックフレームワーク | Next.js |
| 音声処理 | Web Audio API、Tone.js など |
| ビジュアルプログラミング | ノードエディタ系ライブラリ |

ライブラリは積極的に活用する。車輪の再発明を避け、研究・実装の本質に集中する。

## 研究環境

| 役割 | 技術 |
|---|---|
| 実験ノート | Jupyter Notebook（`.ipynb`） |
| 言語 | Python |
| 音声処理 | NumPy / SciPy / librosa など |
| 可視化 | Matplotlib / Plotly など |

## 備考

- プロトタイプと研究環境は独立しており、それぞれ最適な技術を選択する
- 研究で得た知見をプロトタイプに活かすサイクルを意識する
